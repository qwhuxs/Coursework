# System Specification Document (SSD) — Інтернет-магазин сумок

## 1. Технологічний стек
* **Backend:** PHP 8.x (Native).
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript.
* **Database:** MySQL.
* **Сервер:** Apache (налаштування через `.htaccess`).

## 2. Архітектура (MVC)
Проєкт реалізовано за паттерном Model-View-Controller:
* **Models (`/models`):** Обробляють логіку даних (наприклад, `ProductModel.php` отримує список сумок з БД).
* **Views (`/views`):** Містять HTML-код сторінок (картка сумки, головна сторінка, кошик).
* **Controllers (`/controllers`):** Отримують запити від користувача та вирішують, які дані взяти з моделі та яку сторінку показати.

## 3. Маршрутизація
Використовується єдина точка входу — `router.php`. Всі запити перенаправляються через файл `.htaccess`, що дозволяє створювати "чисті" посилання (наприклад, `/product/12` замість `product.php?id=12`).

## 4. Вимоги до оточення
* PHP >= 7.4
* MySQL Server
* Налаштований Virtual Host на веб-сервері.
