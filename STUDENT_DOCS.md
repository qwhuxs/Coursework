# STUDENT_DOCS: Пояснювальна записка до проєкту

## Тема:
**Розробка веб-додатку «Euphoria — Інтернет-магазин сумок» на основі архітектури MVC**

---

## 1. ВСТУП

**Актуальність теми:**  
У сучасному цифровому середовищі інтернет-магазини стали важливою складовою електронної комерції. Торгівля аксесуарами активно переміщується в онлайн-простір, де конкуренція вимагає якісних технічних рішень. Використання архітектурного шаблону MVC (Model–View–Controller) дозволяє створити гнучкий та зручний у супроводі веб-додаток.

**Об’єкт дослідження:**  
Технології розробки веб-додатків.

**Предмет дослідження:**  
Побудова інтернет-магазину на основі MVC з використанням PHP та бази даних MySQL, що забезпечує ефективну взаємодію між користувачем і системою.

**Мета роботи:**  
Розробити інтернет-магазин сумок на мові програмування PHP з використанням архітектури MVC.

**Завдання роботи:**
- Проаналізувати вимоги до функціональності магазину.
- Проєктувати структуру бази даних.
- Реалізувати компоненти MVC.
- Створити інтерфейс користувача та адміністратора.
- Провести тестування системи.

**Практичне значення:**  
Створена система може бути використана як реальний інструмент для малого бізнесу або як база для розробки складніших маркетплейсів.

---

## 2. УМОВНІ ПОЗНАЧЕННЯ ТА СКОРОЧЕННЯ

- **MVC (Model-View-Controller)** — архітектурна модель розділення логіки додатка.  
- **PDO (PHP Data Objects)** — інтерфейс для доступу до баз даних у PHP.  
- **СУБД** — система управління базами даних.  
- **CRUD** — Create, Read, Update, Delete.

---

## 3. РОЗДІЛ 1. ОПИС ТЕХНОЛОГІЧНОГО СТЕКУ

Для реалізації проєкту було обрано стек технологій, що забезпечує баланс між продуктивністю та структурованістю коду:

### Backend (PHP 8.x)
Основна мова розробки. Використання PHP дозволяє динамічно генерувати контент та ефективно взаємодіяти з сервером.

### База даних (MySQL)
Для зберігання даних використано реляційну СУБД. Підключення реалізовано через клас `Database` з використанням паттерна **Singleton**, що забезпечує єдине підключення до БД на весь цикл роботи скрипта.

### Архітектура MVC

- **Models:**  
  Містять SQL-запити (наприклад, `getAllCategories()` у `CategoryModel.php`).

- **Controllers:**  
  Обробляють вхідні HTTP-запити та координують роботу моделей та представлень (наприклад, `CategoryController`).

- **Views:**  
  Шаблони інтерфейсу (HTML/CSS), що відображають дані користувачеві.

### Адміністративна панель
Реалізовано розширений функціонал для:
- керування асортиментом  
- управління замовленнями (зміна статусів: «Очікує», «Виконано» тощо)  
- модерації відгуків
